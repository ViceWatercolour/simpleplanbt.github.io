Project Rot Rots

This is a project the Simple Plan Build Team started. The point of it is so create a War Wars version of every map that is on Overcast Network's rotations.

Select your Map

After selecting the map you would like to see, you will be presented with the list of modified maps.

The number next to the gamemode shows the amount of the maps that are under that category.


NOTE: The gamemodes in the table represent the ORIGINAL gamemode that the map had! All customised maps are DTC (Destroy The Core)!
